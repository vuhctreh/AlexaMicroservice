Instructions:

There are two dependencies:

--------------------------------
github.com/gorilla/mux v1.8.0
github.com/joho/godotenv v1.4.0
--------------------------------

Please install them using either:

--------------------------------
go get github.com/gorilla/mux
go get github.com/joho/godotenv
--------------------------------

OR

--------------------------------
go get ./...
--------------------------------

Although the .env file is included in the ZIP,
you may set your own env files by editing the
appropriate fields in the provided .env, or
make your own with the following format:

--------------------------------
WOLFRAM_API=
AZURE_KEY=
--------------------------------

I understand that this may not be considered a
microservice per say, however, I have made the
decision of compiling a single multi-threaded
router for ease of demonstration.

Thank you!
